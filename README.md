# Blast
blast videogame example

## Консольный клиент
Для запуска консольного клиента надо вызвать: **node view/gameinconsole.js**

## Автотесты
1. Для установки автотестов нужно: **npm install**
2. Для запуска автотестов нужно: **npm test**;
